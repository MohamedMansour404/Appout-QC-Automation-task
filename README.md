# QC Automation Intern Task

## Overview

This task demonstrates manual and automated testing for the SauceDemo (Swag Labs) website.  
It covers the main user flow and validates both functionality and basic user experience.

---

## Scope

The testing focuses on the core flow:

Login → Add to Cart → Checkout → Finish Order

Both positive and negative scenarios were included.

---

## Contents

- Manual test cases (Excel)
- Bug reports with screenshots
- Automation script (Playwright)

---

## Tech Stack

- Playwright
- JavaScript
- Node.js
